What did we learn about remote?

  • Simple google forms to get some answers
  • Only 24% were forced to go remote
  • On scale 1-5 -> rated 4.26 on avg

Remote pros

  • First place: commute time
  • Flexible work hours
  • Own environment can be way better than office one
  • Ability to use work's downtime to do personal tasks
  • More productive
  • More focus
  • More family time

Some problems about remote

(and how to solve them)

  • Internet
  • Socialising
  • Team communication
  • Less focus
  • Life-work balance
  • Worse working environment

Some companies

  • "We don't believe our employees"
  • "Remote now, onsite later"

Remote is not enough

We cannot address those issues with just going remote

Why go distributed?

  • Talent pool
  • "I can work from wherever!"
  • Less office costs, that can be spend on...
  • Company events, gathering people together for awesome time

Distributed? What about timezones?

  • They do exist and can be PitA
  • We make it work with AMER/EMEA/APAC
  • You need to embrace them if you want to go distributed

How to handle async?

Async communication as a first class citizen

  • Use something like slack
  • Don't ping peole unless it's really urgent and important
  • It usually is not that important/urgent, you can wait for an answer/reaction
  • Remember to overcommunicate, write your todo "publicly", etc.

Some things need to stay sync

  • Mail is still a thing
  • Create artefacts
  • Every discussion/decision/meeting should leave something permanent
  • Don't overuse it, you usually don't need to meet for discussion
  • But you NEED to meet for chitchat

Summary

  • There are a lot of pros to remote work
  • Some of the cons are because we were not prepared
  • Others are because we still want it to be local and sync
  • Everybody wins
